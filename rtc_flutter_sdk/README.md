# RTC Flutter SDK

Pure RTC Platform Flutter bridge for Android apps.

This package wraps the Android AAR packaged at `android/local-maven/com/rtcone/rtc-default-sdk/0.1.0/rtc-default-sdk-0.1.0.aar` and exposes it through `RtcFlutterSdk`. It does not depend on any third-party RTC engine package.

## Add To A Flutter App

```yaml
dependencies:
  rtc_flutter_sdk:
    path: ../rtc_flutter_sdk
```

Then start from a short-lived token generated by the dashboard/backend:

```dart
import 'package:rtc_flutter_sdk/rtc_flutter_sdk.dart';

final result = await RtcFlutterSdk.start(
  accessToken: dashboardAccessToken,
  roomId: 'support-room-1',
  speakerOn: true,
);
```

If the token already contains `roomId` or `room_id`, `roomId` can be omitted.

## Events

```dart
RtcFlutterSdk.events.listen((event) {
  switch (event.name) {
    case 'connected':
      break;
    case 'statusChanged':
      break;
    case 'participantCountChanged':
      break;
    case 'error':
      break;
  }
});
```

## Controls

```dart
await RtcFlutterSdk.muteLocalAudio(true);
await RtcFlutterSdk.setSpeakerphoneOn(true);
await RtcFlutterSdk.setLocalVideoEnabled(true);
await RtcFlutterSdk.switchCamera();
await RtcFlutterSdk.setNoiseCancellationEnabled(true);
await RtcFlutterSdk.sendMessage('Hello');
await RtcFlutterSdk.leaveRoom();
await RtcFlutterSdk.release();
```

The plugin requests the Android microphone and camera permissions required by the token before starting the native SDK.
